# container-security-lab
A lab-based collection of container security experiments, covering Docker, Kubernetes, misconfigurations, privilege escalation, and runtime threats.

This repository is a hands-on container security lab designed to:
Demonstrate common Docker and Kubernetes misconfigurations
Explore privilege escalation and attack scenarios in containers
Implement security best practices for containerized workloads
Provide practical examples for learning DevSecOps and cloud-native security
